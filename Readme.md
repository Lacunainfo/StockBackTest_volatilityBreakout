래리윌리엄스 변동성 돌파 전략 백테스트

가장 간단한 형태의 변동성 돌파가격 전략을 모사함

Range = (전날 고가 - 전날 저가) x K
(K는 통상 0.5)
변동성 돌파가격 = 당일 시가 + Range
현재가가 변동성 가격을 돌파시 -> 매수
다음 날 시초가 청산

Input DB 데이터의 형태
(키움 API 에서 OPT100081로 받을 수 있는 데이터의 형태와 같음)

테이블명 : 종목코드
  index     open high   low   close  volume
YYYYMMDD    시가  고가  저가  종가  거래량
